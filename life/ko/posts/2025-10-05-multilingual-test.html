다국어 블로그 지원

이 블로그가 이제 다국어를 지원한다는 소식을 알려드리게 되어 기쁩니다! 네비게이션 바의 언어 전환 버튼을 사용하여 영어와 한국어 사이를 전환할 수 있습니다.

기능

  • 자동 번역: 모든 UI 요소가 자동으로 번역됩니다
  • 언어 설정 유지: 선택한 언어 설정이 저장됩니다
  • 이중 언어 콘텐츠: 포스트를 여러 언어로 작성할 수 있습니다
  • SEO 친화적: 각 언어 버전에 적절한 메타 태그가 있습니다

작동 방식

블로그는 다음과 같은 커스텀 국제화(i18n) 시스템을 사용합니다:

  1. 선호하는 언어 감지
  2. 적절한 번역 로드
  3. 선택한 언어로 콘텐츠 렌더링

언어 전환 버튼을 클릭해서 직접 확인해보세요!
