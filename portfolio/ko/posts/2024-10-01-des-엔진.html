고성능 이산 이벤트 시뮬레이션 엔진

개요

밀리초 미만의 이벤트 처리 지연시간으로 수백만 개체를 처리할 수 있는 Python 기반 이산 이벤트 시뮬레이션 엔진을 개발했습니다. 이 엔진은 네트워크 모델링, 물류 최적화, 에이전트 기반 모델링을 위한 대규모 시뮬레이션을 구동합니다.

기술적 성과

성능 지표

  • 단일 노드에서 초당 1천만 이벤트 처리
  • Calendar Queue로 밀리초 미만 이벤트 스케줄링
  • C 확장과 JIT 컴파일을 통한 100배 속도 향상
  • 분산 처리로 64코어까지 선형 확장성

핵심 기술

  • Python 3.11+ 타입 힌트와 async/await 활용
  • Cython 성능 크리티컬 경로 최적화
  • NumPy 벡터화 연산
  • Ray 분산 시뮬레이션
  • MPI HPC 클러스터 배포

주요 기능

적응형 스케줄러

워크로드 특성에 따라 Calendar Queue, Ladder Queue, Splay Tree 간 자동 전환하는 동적 스케줄러 구현으로 평균 O(1) 복잡도 달성.

Zero-Copy 아키텍처

메모리 맵 파일과 버퍼 프로토콜을 사용한 공유 메모리 시스템 설계로 멀티프로세스 시뮬레이션에서 데이터 복사 오버헤드 제거.

Time Warp 프로토콜

Jefferson의 Virtual Time과 낙관적 동기화 구현으로 자동 롤백과 안티메시지 처리를 통한 효율적인 분산 시뮬레이션 가능.

임팩트

  • 백만 개체 모델의 시뮬레이션 실행 시간을 며칠에서 시간 단위로 단축
  • 물류 네트워크를 위한 실시간 의사결정 지원 가능
  • 핵심 컴포넌트 오픈소스화로 GitHub 500+ 스타 획득
  • 3개 연구 기관에서 에이전트 기반 모델링에 채택

코드 예시

from simulation import Engine, CalendarQueue

engine = Engine(scheduler=CalendarQueue())
engine.schedule(time=0.0, event=InitializeSimulation())
engine.run(until=86400.0)  # 24시간 시뮬레이션 시간 실행
