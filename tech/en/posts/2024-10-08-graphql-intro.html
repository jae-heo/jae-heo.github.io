Introduction to GraphQL

GraphQL provides a more efficient alternative to REST APIs.

What is GraphQL?

A query language for APIs that allows clients to request exactly what they need.

Schema Definition

type User {
  id: ID!
  name: String!
  email: String!
  posts: [Post!]!
}

type Post {
  id: ID!
  title: String!
  content: String!
  author: User!
}

Queries

query GetUser($id: ID!) {
  user(id: $id) {
    name
    posts {
      title
    }
  }
}

Advantages

  • No over-fetching
  • No under-fetching
  • Strong typing
  • Self-documenting
