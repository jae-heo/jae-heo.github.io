수백만 개체를 다루는 이산 이벤트 시뮬레이션에서 메모리 관리는 성능의 핵심입니다. Python의 메모리 모델을 깊이 이해하고 최적화하는 방법을 살펴봅니다.

Python 메모리 할당자 계층 구조

CPython은 세 가지 레벨의 메모리 할당을 사용합니다:

import ctypes
import sys
from memory_profiler import profile

class PyObjectHead(ctypes.Structure):
    """CPython 객체 헤더 구조"""
    _fields_ = [
        ('ob_refcnt', ctypes.c_ssize_t),
        ('ob_type', ctypes.c_void_p),
    ]

def analyze_memory_layout(obj):
    """객체의 실제 메모리 레이아웃 분석"""
    address = id(obj)
    py_object = ctypes.POINTER(PyObjectHead).from_address(address)

    return {
        'address': hex(address),
        'refcount': py_object.contents.ob_refcnt,
        'type_ptr': hex(py_object.contents.ob_type),
        'size': sys.getsizeof(obj)
    }

# 메모리 풀 사전 할당
class SimulationMemoryPool:
    def __init__(self, entity_count: int = 1_000_000):
        # Pymalloc 아레나 최적화를 위한 청크 크기
        self.POOL_SIZE = 256 * 1024  # 256KB chunks
        self.entity_slots = []

        # 메모리 프래그멘테이션 방지를 위한 사전 할당
        for _ in range(entity_count // 1000):
            chunk = bytearray(self.POOL_SIZE)
            self.entity_slots.append(chunk)

Zero-Copy 데이터 구조 구현

시뮬레이션 상태 전송에서 복사 오버헤드를 제거:

import numpy as np
from multiprocessing import shared_memory
import struct

class SharedSimulationState:
    """프로세스 간 zero-copy 공유 상태"""
    def __init__(self, n_entities: int, state_size: int = 64):
        self.n_entities = n_entities
        self.state_size = state_size

        # 공유 메모리 생성
        self.shm = shared_memory.SharedMemory(
            create=True,
            size=n_entities * state_size
        )

        # numpy 배열로 직접 매핑
        self.states = np.ndarray(
            (n_entities, state_size // 8),  # 8 bytes per float64
            dtype=np.float64,
            buffer=self.shm.buf
        )

    def update_entity(self, entity_id: int, state_vector: np.ndarray):
        """Lock-free 상태 업데이트"""
        # 메모리 뷰를 통한 직접 쓰기
        self.states[entity_id, :] = state_vector

    def broadcast_state(self, target_processes: list):
        """Zero-copy 브로드캐스트"""
        # 공유 메모리 이름만 전송
        return self.shm.name

슬롯 클래스와 slots 최적화

class OptimizedEntity:
    """메모리 효율적인 엔티티 클래스"""
    __slots__ = ('_id', '_state', '_next_event', '_attributes')

    # 클래스 레벨 객체 풀
    _pool = []
    _pool_size = 10000

    def __new__(cls):
        if cls._pool:
            # 재사용 가능한 인스턴스 반환
            instance = cls._pool.pop()
            return instance
        return super().__new__(cls)

    def __del__(self):
        # 객체를 풀로 반환
        if len(self._pool) < self._pool_size:
            # 상태 초기화
            self._id = None
            self._state = None
            self._next_event = None
            self._attributes = None
            self._pool.append(self)

메모리 맵 파일 활용

대용량 시뮬레이션 데이터의 효율적 관리:

import mmap
import os
import pickle

class SimulationCheckpoint:
    """메모리 맵을 활용한 체크포인트 시스템"""

    def __init__(self, checkpoint_file: str, size_gb: int = 10):
        self.file_path = checkpoint_file
        self.size = size_gb * 1024 * 1024 * 1024

        # 파일 생성 및 크기 할당
        with open(self.file_path, 'wb') as f:
            f.seek(self.size - 1)
            f.write(b'\0')

        self.file = open(self.file_path, 'r+b')
        self.mmap = mmap.mmap(
            self.file.fileno(),
            0,  # 전체 파일 매핑
            access=mmap.ACCESS_WRITE
        )

        self.header_size = 1024
        self._write_header()

    def save_state(self, simulation_time: float, entities: list):
        """Incremental checkpoint 저장"""
        # 현재 위치 찾기
        offset = struct.unpack('Q', self.mmap[8:16])[0]

        # 시뮬레이션 시간 저장
        self.mmap[offset:offset+8] = struct.pack('d', simulation_time)
        offset += 8

        # 엔티티 상태 직렬화 (pickle protocol 5 for out-of-band data)
        for entity in entities:
            data = pickle.dumps(entity, protocol=5)
            size = len(data)

            self.mmap[offset:offset+4] = struct.pack('I', size)
            self.mmap[offset+4:offset+4+size] = data
            offset += 4 + size

        # 오프셋 업데이트
        self.mmap[8:16] = struct.pack('Q', offset)

Garbage Collection 튜닝

시뮬레이션 중 GC 일시정지 최소화:

import gc
import time

class GCOptimizedSimulation:
    def __init__(self):
        # GC 임계값 조정
        gc.set_threshold(700, 10, 10)

        # 세대별 수집 빈도 감소
        self.gc_gen0_threshold = 50000
        self.object_count = 0

    def run_simulation_step(self):
        """GC 친화적인 시뮬레이션 스텝"""
        # 수동 GC 제어
        self.object_count += 1

        if self.object_count % self.gc_gen0_threshold == 0:
            # Generation 0만 수집
            gc.collect(0)

        # 크리티컬 섹션에서 GC 비활성화
        gc.disable()
        try:
            # 시간 민감한 시뮬레이션 로직
            self._process_critical_events()
        finally:
            gc.enable()

    def _process_critical_events(self):
        """GC 없이 실행되어야 하는 크리티컬 이벤트"""
        start = time.perf_counter_ns()

        # 이벤트 처리 로직
        # ...

        elapsed = time.perf_counter_ns() - start
        if elapsed > 1_000_000:  # 1ms 초과
            print(f"Warning: Critical section took {elapsed/1e6:.2f}ms")

메모리 프로파일링과 최적화

from memory_profiler import profile
import tracemalloc

class MemoryProfiler:
    def __init__(self):
        tracemalloc.start()
        self.snapshots = []

    def take_snapshot(self, label: str):
        snapshot = tracemalloc.take_snapshot()
        self.snapshots.append((label, snapshot))

        # 상위 메모리 사용 통계
        top_stats = snapshot.statistics('lineno')[:10]

        print(f"\n=== {label} ===")
        for stat in top_stats:
            print(f"{stat.traceback.format()[0]}")
            print(f"  Size: {stat.size / 1024 / 1024:.1f} MB")

    def compare_snapshots(self, idx1: int, idx2: int):
        """스냅샷 간 메모리 차이 분석"""
        _, snapshot1 = self.snapshots[idx1]
        label2, snapshot2 = self.snapshots[idx2]

        top_stats = snapshot2.compare_to(snapshot1, 'lineno')[:10]

        print(f"\n=== Changes to {label2} ===")
        for stat in top_stats:
            print(f"{stat.traceback.format()[0]}")
            print(f"  Change: {stat.size_diff / 1024 / 1024:+.1f} MB")

# 사용 예시
profiler = MemoryProfiler()
profiler.take_snapshot("Initial")

# 시뮬레이션 실행
simulation = GCOptimizedSimulation()
for _ in range(10000):
    simulation.run_simulation_step()

profiler.take_snapshot("After 10k steps")
profiler.compare_snapshots(0, 1)
