재영의 테크 블로그 소개

제 기술 블로그에 오신 것을 환영합니다! 이곳은 프로그래밍 언어 심화 학습부터 최신 도구와 프레임워크 탐구까지, 끊임없이 진화하는 기술 세계를 통한 제 여정을 공유하는 곳입니다.

주요 콘텐츠

🚀 소프트웨어 개발

현대적인 개발 방식, 디자인 패턴, 아키텍처 결정을 탐구합니다. 마이크로서비스부터 모놀리스까지, REST부터 GraphQL까지, 소프트웨어를 움직이게 하는 것들을 깊이 있게 다룹니다.

🤖 AI & 머신러닝

AI 혁명이 진행 중입니다. LLM, 머신러닝 프레임워크, 그리고 일상 개발에서의 AI 실용적 응용에 대한 경험을 기록합니다.

⚡ 웹 기술

React부터 Vue까지, Node.js부터 Deno까지, 현대 웹을 구동하는 도구들을 탐구합니다. 특히 성능 최적화와 개발자 경험에 관심이 많습니다.

🛠️ DevOps & 클라우드

코드형 인프라, 컨테이너화, CI/CD 파이프라인 - 클라우드에서 확장 가능한 시스템 구축 및 유지 관리에 대해 씁니다.

📱 모바일 개발

네이티브, 하이브리드, 또는 PWA? 다양한 모바일 개발 접근 방식을 탐구하고 크로스 플랫폼 애플리케이션 구축에 대한 인사이트를 공유합니다.

블로그를 운영하는 이유

공개적으로 학습하는 것을 믿습니다. 모든 포스트는 다음을 위한 기회입니다:

  • 학습 여정 문서화
  • 커뮤니티와 지식 공유
  • 동료 개발자들과 연결
  • 기술 문서 작성 실력 향상
  • 나를 가르쳐준 커뮤니티에 보답

나의 기술 여정

저는 재영, 열정적인 소프트웨어 개발자입니다:

  • 프론트엔드: React, Vue, TypeScript, Next.js
  • 백엔드: Node.js, Python, Go, PostgreSQL, MongoDB
  • 클라우드: AWS, Google Cloud, Docker, Kubernetes
  • 도구: Git, VS Code, Vim, 터미널 마법

호기심과 간단한 "Hello, World!"로 코딩 여정을 시작했습니다. 오늘날 저는 복잡한 시스템을 구축하며 모든 도전을 즐기고 있습니다.

기술 스택

이 블로그 자체가 처음부터 무언가를 만들기 좋아하는 제 성향의 증거입니다:

  • 커스텀 정적 사이트 생성기 - 순수 Node.js로 구축
  • 마크다운 파서 - 콘텐츠 유연성을 위해
  • 바닐라 JavaScript - 심플하고 빠르게
  • 다국어 지원 - 기술은 글로벌하니까
  • GitHub Actions - 자동화된 배포 파이프라인
